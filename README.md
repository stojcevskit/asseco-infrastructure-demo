# Asseco Infrastructure Demo (2025)

This project demonstrates a fully automated infrastructure deployment using **Terraform** and **Docker**.

## Architecture
- **Nginx Proxy**: Web entry point.
- **MariaDB**: Database with persistent storage.
- **Netdata**: Real-time monitoring.
- **ELK Stack**: Centralized logging (Elasticsearch & Kibana).

## How to run
1. Run `terraform init`
2. Run `terraform apply`
